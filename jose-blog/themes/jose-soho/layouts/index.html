About

I'm a Ph.D. student at the University of New Mexico in the Computer Science Department advised by Prof. Deepak Kapur working on formal methods and computer algebra. I'm interested in software verification, programming languages, commutative algebra, and non-classical logics.

I obtained a bacherlor's degree in electronics engineering from Universidad de las Americas Puebla advised by Prof. Mauricio Osorio Galindo, working on the paraconsistent logic \(C_1\).

My CV can be accessed here.

Software

  • AXDInterpolator: implementation of a quantifier-free interpolation algorithm for the theory of arrays and index theories (i.e. total order, integer difference logic, linear integer arithmetic).
  • EUFInterpolator: implementation of a quantifier-free interpolation algorithm for the theory of equality and uninterpreted functions.
  • Z3-Inter-Plus: fork of the Z3 SMT solver including additional utilities to process common symbols, context processing, and a custom rewriter to relax QF_IDL expressions to QF_TO expressions.
  • z3-proofs2latex: A Python script to export Z3 proofs to \( \LaTeX \).
  • QuickTex: Tool to automatically setup \( \LaTeX \) projects.
